## Технологии

Проект разработан с использованием следующих технологий:

- HTML5
- CSS

## Как использовать (гайд)
1.
```
git clone [https://github.com/alkun21/shop_glasses.git]
```
2.
```
cd shop_glasses 
```
3.
```
start index.html
```
4. Просматривайте очки в магазине
